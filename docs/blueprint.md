# **App Name**: Shabistan Hub

## Core Features:

- Interactive Menu: A digital, interactive version of Shabistan's menu, displaying items, descriptions, and prices as detailed in the prompt.
- Order/Reservation Form: A form allowing customers to submit pre-orders or make small group reservations with the option to save reservations to the Cloud SQL database.
- Gallery Showcase: A visually appealing gallery showcasing high-quality images of Shabistan's offerings and ambiance.
- AI Menu Recommendation: Generative AI that acts as a "tool" to provide personalized menu recommendations based on user preferences (e.g., dietary restrictions, past orders) to suggest relevant items.
- Customer Review Display: Showcase customer testimonials and reviews on the website to build trust and credibility.

## Style Guidelines:

- Primary color: Brand red (#C9252C) from the "THE MEETING POINT" tagline, used for main accents and calls to action. Evokes excitement and appetite.
- Background color: Light cream (#FFF9EB) with a slightly textured feel, serving as the main background. A low-saturation version of the logo's caramel/waffle tone ensures visual harmony and creates a cozy atmosphere.
- Accent color: Fresh green (#2E7D32) from drink and menu borders. An analogous color to the red that also strongly contrasts it in saturation and brightness, conveying freshness and a natural vibe.
- Headings font: 'Playfair', a friendly, slightly classic serif that mirrors the logo's vibe. Evokes sophistication and invites attention.
- Body font: 'Inter', a clean, modern sans-serif for easy readability. It is unobtrusive, accessible and clear.
- Use outlined icons representing menu items (cones, cups, fries, waffles) to add a playful touch.
- Rounded corners, soft shadows, and playful elements for a casual, inviting atmosphere.
- Subtle hover effects on buttons and cards, such as slight scale-up and color darkening, to provide visual feedback and interactivity.
